# 📚 Smart Library Management System

A modern web-based Library Management System designed to manage books, students, issue/return records, and library operations efficiently.

---

## 🚀 Features

- 📖 Add, update, delete, and search books  
- 👨‍🎓 Manage student/member records  
- 🔄 Issue and return books  
- 📅 Track due dates and availability  
- 🔍 Search functionality for books and users  
- 📊 Dashboard for library statistics  
- 🔐 User authentication  
- 🌐 Responsive frontend UI  

---
